# UmaMusume Team Trials Tracker - Frontend

Este repositorio contiene el frontend de la aplicación UmaMusume Team Trials Tracker, construido con Next.js y Tailwind CSS.

## Despliegue

Recomendado para Vercel. Configura la variable de entorno `NEXT_PUBLIC_API_URL` con la URL de tu backend.

## Instalación

```bash
npm install
npm run dev
```

## Variables de entorno

- `NEXT_PUBLIC_API_URL`: URL del backend (ejemplo: https://backend-url/api)
